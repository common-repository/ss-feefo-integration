== Description ==

SS Feefo Integration is a plugin that allows you to export the products and sales in a XML format that can be used to import in Feefo website. This plugin requires the WooCommerce plugin. You also need to provide the 'Merchant Identifier' prodivded by Feefo. 
<br>You can get the merchant identifier from your Feefo account.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Feefo Admin screen to configure the plugin
